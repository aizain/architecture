# bower

> 简介
```
官方网站：
https://bower.io/

官方说明：
Web sites are made of lots of things — frameworks, libraries, assets, and utilities. Bower manages all these things for you.

听说是 twitter 搞的，用来模块化管理前端的依赖

本文章使用版本 1.8.0+
```

> 安装
```
npm install -g bower
```

> 使用
```
// 在当前目录下，建立bower项目，并且下载包
bower install <package>

// 下载git项目
bower install https://github.com/xxx/xxx.git
```
